# 6. 간단한 가계부

- **난이도**: 레벨 2 (초급)

1. 데이터를 어떤 형식으로 저장할까요? (CSV 파일, JSON 파일, 텍스트 파일)

## 💡 학습 팁

### 대화형 개발의 핵심

- ✅ **"물어봐줘"**: Codex가 능동적으로 질문하게 만듦
- ✅ **단계적 구체화**: 처음부터 완벽한 요구사항 불필요
- ✅ **선택과 결정**: 개발 과정에서 의사결정 연습

### 이 프로젝트에서 배울 수 있는 것

- **CSV 파일**: `csv.writer()`, `csv.reader()`로 데이터 읽기/쓰기
- **데이터 집계**: 합계, 평균, 비율 계산
- **날짜 처리**: `datetime` 모듈로 날짜 비교, 기간 계산
- **리스트 컴프리헨션**: 데이터 필터링
- **포맷팅**: 금액 표시 (천 단위 쉼표)

### 추가 도전 과제

- 정기 지출 자동 등록 (월세, 구독료)
- 영수증 사진 경로 저장
- 고정 지출 vs 변동 지출 분류
- 연간 리포트 생성
